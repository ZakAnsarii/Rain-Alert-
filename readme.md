☔ Rain Alert ☔

A Python automation project that checks the OpenWeather forecast every morning and sends an email reminder if rain is expected during office hours.

I built this project to learn the basics of working with APIs and automation by creating a simple real-world application. It also helped me learn Git, GitHub, GitHub Actions, environment variables, and SMTP email automation.

How It Works

Every morning, GitHub Actions runs the script, checks today's weather forecast using the OpenWeather API, and sends an email if rain is expected during office hours.
